# EMPG: Entropy-Modulated Policy Gradients

- **ID**: 2509-empg
- **Link**: [alphaxiv](https://www.alphaxiv.org/abs/2509.09265) · [知乎](https://zhuanlan.zhihu.com/p/1950709316148494600)

## TL;DR

**核心发现**（Proposition 1）：softmax policy gradient 的 norm 与 entropy **单调耦合**：$\mathbb{E}[\|\nabla \log \pi\|^2] = 1 - \exp(-H_2(\pi))$。后果：confident correct step 梯度小（学不够快）、uncertain step 梯度大（训练不稳定）。**EMPG 用 step 级 entropy 重新标定梯度幅度**，并加一个 "future clarity bonus" 引导 agent 走向更可预测的状态。

## Method

### 两件套

$$A_{\text{mod}}(i,t) = \underbrace{A^{(i)} \cdot g(H_t^{(i)})}_{\text{self-calibrating gradient scaling}} + \underbrace{\zeta \cdot f(H_{t+1}^{(i)})}_{\text{future clarity bonus}}$$

### 1. Self-Calibrating Gradient Scaling $g(H_t)$

流程（假设 batch 有 12 个 step）：

1. 对每个 step，计算该 step 内所有 token 的平均 entropy $H_t$
2. Min-max 归一化到 $[0, 1]$：

$$H_{\text{norm}} = \frac{H_t - \min}{\max - \min + \epsilon}$$

3. 计算缩放：

$$g(H_t) = \frac{\exp(-k \cdot H_{\text{norm},t})}{\frac{1}{N}\sum_{\text{all N}} \exp(-k \cdot H_{\text{norm}})}$$

- Confident step ($H_{\text{norm}}$ 小) → $g > 1$（放大）
- Uncertain step → $g < 1$（衰减）
- **均值归一化** 保证总 signal 量不变——只做重分配

### 2. Future Clarity Bonus $f(H_{t+1})$

$$\zeta \cdot \exp(-k' \cdot H_{\text{norm},t+1})$$

**直觉**：这一步之后 agent 在下一步变得很"确定"（低 entropy），说明这一步选对了方向。

例子：

| 场景 | $H_{t+1}$ | Bonus |
|---|---|---|
| Search 到高相关文档 → 下一步很 confident 提取段落 | 低 | 高 |
| Search 到无关结果 → 下一步完全 confused | 高 | 低 |

## Key Results

| 任务 | 模型 | EMPG 增量 |
|---|---|---|
| ALFWorld | Qwen2.5-1.5B/7B | +1.6 ~ +8.1 |
| WebShop | Qwen2.5-1.5B/7B | +0.6 ~ +3.7 |
| Deep Search | Qwen2.5-32B | +3.3 (overall) |

## Takeaway

1. **Proposition 1 揭示的问题很根本**——所有 softmax policy 都有这个病，不止 agent
2. **"未来清晰度" 是 reward shaping 的新视角**：不直接奖励结果，而是奖励"导向可解释状态"
3. **可深入方向**（论文建议）：
   - 第 i 轨迹的每个 token 共用 advantage $A^{(i)}$，但按 token entropy 缩放
   - 下一个 token 熵低，就给上一个 token bonus
